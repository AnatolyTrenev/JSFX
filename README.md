Installation (via ReaPack)

1. Open REAPER
2. Extensions → ReaPack → Import repositories
3. Paste this URL:

https://github.com/AnatolyTrenev/JSFX/raw/master/index.xml

4. Synchronize packages
5. Install **Creamy Wide**

The plugin will appear in:
Effects / Trenev /

## Included plugins

- Creamy Wide
- Wilder (comming soon)
